This directory is for non-official MWCNT multi-seed validation development work.

Nothing here is part of the maintained validation test lane.

Current purpose:
- generate deterministic MWCNT geometries with the dev-selected dynamic RSA path,
- run cumulative 1, 3, and 6-seed averaged validation studies in `nrss-dev`,
- reuse the maintained MWCNT reduction and plot panel formatting for direct comparison.

Primary script:
- `run_mwcnt_multiseed_study.py`

Outputs are written under:
- `test-reports/mwcnt-multiseed-dev/`
